# Fatigue Management App for Cancer Survivors
    
암생존자는 암 진단 이후부터 말기 호스피스 단계 이전까지, 더 이상 적극적인 치료가 어려운 시기를 말합니다. 최근 암 치료 기술의 발전, 암 예방 노력, 조기 검진의 확산으로 국내 암생존율이 꾸준히 상승하고 있으며, 암생존자의 수도 지속적으로 증가하고 있습니다.
하지만 암생존자는 치료 후에도 신체적, 정신적, 사회적 어려움을 겪습니다. 그 중에서도 암 관련 피로는 대부분의 암생존자들이 겪는 증상으로, 일상생활로 복귀하는 데 가장 큰 장애물 중 하나입니다.

**아미타민 팀**은 암생존자들이 이러한 피로를 스스로 관리할 수 있도록 돕기 위해 **생활 밀착형 앱**을 제작하였습니다. 이 앱은 암생존자들이 일상생활로 성공적으로 복귀하고, 더 나은 삶의 질을 유지할 수 있도록 지원하는 데 목적이 있습니다.
